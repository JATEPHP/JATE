The jRequire statement includes and evaluates the specified file using relative path.<br>
```java
void jRequire ( String $_path )
```

## Parameters
* **$_path**: The path of the file you want to include.

## Return Values

## Examples
```php
<?php
  jRequire("../Module01.php");
?>
```
